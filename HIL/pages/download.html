下载说明

HIL平台提供两种获取方式：直接安装版（一键安装，开箱即用）和Release版本（源码压缩包，适合自行编译部署）。软件已内置Python环境，可直接运行使用。但推荐做法是自行在电脑上安装好对应的Conda环境和依赖，以获得更好的兼容性和性能。下载完成后，请参考环境配置指南完成环境搭建。

直接安装版（分卷压缩）

适用于 Windows 10/11 (64-bit) — 共5个分卷，解压后得到安装程序

下载全部分卷 查看环境配置教程

Release 版本

适用于 Windows 10/11 (64-bit) — 源码压缩包，灵活部署

下载 Release 压缩包 查看环境配置教程

安全提示：下载的 .exe 文件可能会触发浏览器的安全警告，这是正常现象。请点击"保留"或"继续下载"完成下载。下载完成后，建议使用杀毒软件扫描文件后再运行安装程序。
