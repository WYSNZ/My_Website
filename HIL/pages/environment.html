重要提示：HIL软件已内置Python运行环境，配置CUDA和PyTorch后可直接运行但不推荐使用。推荐做法是自行在电脑上安装好对应的Conda环境和依赖，以获得更好的兼容性和性能表现。以下教程将指导您完成完整的环境配置。

前置步骤：检查GPU驱动CUDA兼容性

HIL平台使用PyTorch自带的CUDA 12.4进行GPU加速，无需单独安装CUDA Toolkit。但您的NVIDIA显卡驱动必须支持CUDA 12.4及以上版本。在开始环境配置前，请先检查驱动兼容性。

1
检查当前驱动CUDA版本

打开命令提示符或PowerShell，运行以下命令：

nvidia-smi

在输出信息中找到 "CUDA Version" 一行，例如：

| NVIDIA-SMI 555.85 Driver Version: 555.85 CUDA Version: 12.5 |
2
判断是否需要更新驱动

如果显示的 CUDA Version ≥ 12.4，则驱动兼容，可以继续下一步。

如果CUDA Version < 12.4（例如显示 11.8），则需要更新NVIDIA显卡驱动。请访问NVIDIA官网下载并安装最新驱动：

NVIDIA 驱动下载页面（点击进入官网）

选择您的显卡型号和操作系统下载最新驱动

注意：驱动更新后需要重启计算机才能生效。安装完成后再次运行nvidia-smi确认CUDA版本已更新到12.4以上。

第一步：安装Miniconda

Conda 是一个开源的包管理系统和环境管理系统。我们推荐使用Miniconda（Anaconda的精简版），它只包含conda和Python基础组件，体积更小。

1
下载Miniconda

访问Miniconda官方下载页面，选择Windows 64位安装包：

Miniconda 官方下载（点击进入）

选择Windows 64-bit安装包
2
安装Miniconda
  1. 运行下载的 .exe 安装程序
  2. 点击 "Next" 继续
  3. 接受许可协议
  4. 选择 "Just Me"（推荐）或 "All Users"
  5. 选择安装路径（建议默认路径）
  6. 重要：勾选 "Add Miniconda3 to my PATH environment variable"
  7. 点击 "Install" 开始安装，等待完成

环境变量提示：如果没有勾选添加到PATH，需要手动添加以下路径到系统环境变量：
C:\Users\你的用户名\miniconda3
C:\Users\你的用户名\miniconda3\Scripts
C:\Users\你的用户名\miniconda3\Library\bin

3
验证Conda安装

打开新的命令提示符或PowerShell窗口，运行：

conda --version

如果显示conda版本号，说明安装成功。

第二步：创建并配置Conda环境

我们提供了预配置好的环境文件，您可以选择方式一（推荐）：使用environment.yml一键创建，或方式二：逐步手动安装。两种方式的效果完全一致。

方式一（推荐）
使用environment.yml一键创建环境

下载我们提供的 environment.yml 文件后，打开命令行并切换到文件所在目录，运行：

conda env create -f environment.yml

这会自动创建一个名为 hil 的环境，并安装所有必需的依赖包（包括PyTorch 2.4.0 + CUDA 12.4）。您也可以使用 -n 自定义名称 来指定个性化的环境名：

conda env create -f environment.yml -n my_hil_env

安装完成后激活环境：

conda activate hil

注意：执行上述命令时，请确保当前终端的工作目录与文件所在位置一致（使用 cd 命令切换），否则请使用绝对路径：
conda env create -f C:\Users\你的用户名\Downloads\environment.yml

点击下载 environment.yml

Conda环境配置文件（含所有依赖）

点击下载 requirements.txt

Python pip依赖清单

个性化环境名：默认创建的环境名为 hil，如需自定义名称，可使用 conda env create -f environment.yml -n 你的环境名。

方式二
逐步手动创建环境

步骤1：创建新的Conda环境（Python 3.8）。您可以将 hil 替换为您喜欢的任意名称：

conda create -n hil python=3.8 -y

步骤2：激活环境：

conda activate hil

步骤3：安装PyTorch（GPU版本，自带CUDA 12.4）：

conda install pytorch==2.4.0 torchvision==0.19.0 torchaudio==2.4.0 pytorch-cuda=12.4 -c pytorch -c nvidia -y

步骤4：安装其他依赖包（使用requirements.txt）：

pip install -r requirements.txt

注意：执行上述命令时，请确保当前终端的工作目录与 requirements.txt 文件所在位置一致，否则请使用绝对路径：
pip install -r C:\Users\你的用户名\Downloads\requirements.txt

第三步：验证GPU支持

1
确认PyTorch能识别GPU

在已激活的conda环境中运行以下Python代码：

python -c "import torch; print(f'CUDA available: {torch.cuda.is_available()}'); print(f'CUDA version: {torch.version.cuda}'); print(f'Device count: {torch.cuda.device_count()}'); print(f'Device name: {torch.cuda.get_device_name(0) if torch.cuda.is_available() else \"N/A\"}')"

预期输出示例：

CUDA available: True CUDA version: 12.4 Device count: 1 Device name: NVIDIA GeForce RTX 4060

如果输出显示 CUDA available: True，说明PyTorch已正确识别GPU，环境配置完成！

环境配置完成！您已成功配置好HIL平台所需的Python开发环境。现在可以返回使用教程开始使用平台了。如有疑问，请查看常见问题。
